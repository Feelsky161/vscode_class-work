Основные теги HTML
Структурные теги:
html
<!DOCTYPE html> <!-- Объявление типа документа, обязательно в начале файла -->
<html> <!-- Корневой элемент всего HTML-документа -->
    <head> <!-- Содержит метаинформацию о странице -->
        <title> <!-- Заголовок страницы, отображается в вкладке браузера -->
    </head>
    <body> <!-- Основное содержимое веб-страницы -->
Текстовые элементы:
html
<h1>...</h6> <!-- Заголовки разного уровня важности (h1 - самый важный) -->
<p> <!-- Абзац текста -->
<br> <!-- Перенос строки -->
<hr> <!-- Горизонтальная разделительная линия -->
Списки:
html
<ul> <!-- Неупорядоченный (маркированный) список -->
    <li> <!-- Элемент списка -->
<ol> <!-- Упорядоченный (нумерованный) список -->
    <li> <!-- Элемент списка -->
Ссылки и медиа:
html
<a href="#">ссылка</a> <!-- Гиперссылка -->
описание <!-- Изображение с альтернативным текстом -->
<video> <!-- Контейнер для видео -->
<audio> <!-- Контейнер для аудио -->
Формы:
html
<form> <!-- Контейнер для формы -->
    <input type="text|password|email"> <!-- Поле ввода -->
    <textarea> <!-- Многострочное текстовое поле -->
    <button> <!-- Кнопка -->
    <select> <!-- Выпадающий список -->
Основные свойства CSS
Типографика:
css
font-family: Arial, sans-serif; /* Семейство шрифтов */
font-size: 16px; /* Размер шрифта */
color: #000; /* Цвет текста */
text-align: center; /* Выравнивание текста */
Размеры и отступы:
css
width: 100%; /* Ширина элемента */
height: auto; /* Высота элемента */
margin: 10px; /* Внешние отступы */
padding: 20px; /* Внутренние отступы */
Позиционирование:
css
position: static|relative|absolute; /* Тип позиционирования */
top: 0; /* Смещение сверху */
left: 0; /* Смещение слева */
z-index: 1; /* Порядок наложения элементов */
Flexbox:
css
display: flex; /* Активация flex-контейнера */
justify-content: center; /* Выравнивание по горизонтали */
align-items: center; /* Выравнивание по вертикали */
Box model:
css
border: 1px solid #000; /* Граница элемента */
border-radius: 5px; /* Скругление углов */
box-shadow: 0 2px 5px rgba(0,0,0,0.1); /* Тень элемента */
Селекторы CSS
Базовые:
css
.class /* Выбирает элементы по классу */
#id /* Выбирает элемент по идентификатору */
element /* Выбирает элементы по типу */
Комбинированные:
css
.parent > .child /* Прямой дочерний элемент */
.class1.class2 /* Элементы с несколькими классами */
element.class /* Элемент с определенным классом */
Псевдоклассы:
css
:hover /* Состояние при наведении */
:focus /* Состояние при фокусе */
:active /* Состояние при нажатии */
Медиа-запросы
css
@media screen and (max-width: 768px) {
    /* Стили для экранов шириной до 768px */
}

@media print {
    /* Стили для печати */
}
Препроцессоры (SCSS)
Переменные:
scss
$primary-color: #007bff; /* Определение переменной цвета */
$font-size: 16px; /* Определение переменной размера шрифта */
Вложенные правила:
scss
.container {
    width: 100%;

    .header {
        background: $primary-color; /* Использование переменной */
    }
}
